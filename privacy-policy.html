Privacy Policy

Personal Health Data Extractor Chrome Extension

Last updated: January 2025

Important: This extension is not affiliated with, endorsed by, or connected to Function Health. It is an independent tool created to help users export their own biomarker data for personal use.

Overview

This Privacy Policy describes how the Personal Health Data Extractor Chrome extension handles your information. We are committed to protecting your privacy and ensuring transparency about our data practices.

Data Collection

We do not collect, store, or transmit any personal data to external servers.

The extension operates entirely within your browser and does not send any of your biomarker data to external services, third parties, or the extension developers.

Data Usage

The extension can be ussed to read your Function Health biomarker data solely to:

  • Extract and format your biomarker results from the Function Health website
  • Organize data by medical categories for easier analysis
  • Export data in various formats (CSV, JSON, table) for your personal use

Data Storage

The extension does not permanently store your biomarker data. Data is only held temporarily in browser memory during the extraction and export process. Once you close the extension or navigate away, this data is cleared.

Data Sharing

Zero data sharing: No data is shared with third parties, external services, or the extension developers.

Your biomarker data remains completely private and under your control at all times.

Function Health Integration

The extension uses your existing Function Health login session to access your biomarkers. The extension:

  • Does not store or handle your login credentials
  • Does not modify or interfere with your Function Health account
  • Only reads biomarker data that you already have access to
  • Requires you to be properly logged in to Function Health

Technical Details

Local Processing: All data processing occurs locally within your Chrome browser.

No External Servers: The extension does not communicate with any servers other than Function Health's API.

Session-Based: Uses your existing Function Health authentication session.

Temporary Storage: Data is only held in memory during active use.

Open Source Transparency

This extension is open source, providing complete transparency about data handling practices. The full source code is available for review at:

https://github.com/Greenband1/biomarker_chrome_extension

User Rights

Since we don't collect or store your data, there is no personal data for us to modify or delete. You maintain full control over your biomarker data at all times.

Security

The extension follows Chrome extension security best practices:

  • Minimal permissions requested
  • Content Security Policy compliance
  • No eval() or unsafe code execution
  • Secure API communication with Function Health

Updates to This Policy

We may update this privacy policy from time to time. Any changes will be posted on this page and, if significant, communicated through the Chrome Web Store listing.
